📋 תקציר מנהלים

GEO (Generative Engine Optimization) — האופטימיזציה לתשובות AI. כש-40% מקוני B2B כבר חוקרים ב-ChatGPT/Gemini/Perplexity, מי שלא מופיע בתשובות AI — פשוט לא קיים.

Uranus Invest עשתה עבודה טובה בהקמת תשתית (schemas, llms.txt, תוכן מקצועי), אבל בעיה קריטית אחת מונעת מכל זה לעבוד: robots.txt חוסם את כל ה-AI crawlers. זה כמו לשים שלט מקצועי על חנות נעולה.

✅ מה כבר קיים ועובד

  • ✅ llms.txt — קובץ מפורט ומותאם אישית (Yoast). כולל עמודים, פוסטים, דפי נחיתה, סקירות שוק, קטגוריות. מצוין.
  • ✅ Organization Schema (Yoast) — שם, לוגו, alternateName בעברית, sameAs (Facebook, YouTube, Google Maps)
  • ✅ FAQPage Schema — שאלות ותשובות מובנות על דף הבית (6 שאלות)
  • ✅ Product Schema + AggregateRating — דירוג 4.9/5 מ-357 ביקורות. אות סמכות חזק.
  • ✅ BreadcrumbList + WebSite + SearchAction — מבנה אתר נגיש למנועים
  • ✅ Meta Description + OG Tags — מלאים, בעברית, עם תמונה
  • ✅ Sitemap — 7 sitemaps (דפים, פוסטים, דפי נחיתה, מאמרים, משרות, סקירות שוק, קטגוריות)
  • ✅ תוכן HTML server-side — 30+ מופעי תוכן עברי ב-HTML source (לא רק JavaScript). AI crawlers יכולים לקרוא.
  • ✅ ~17 פוסטים מקצועיים — מידע מקצועי, השקעות קריפטו, סקירות שוק
  • ✅ דף אודות — תום בוקאי, רקע צבאי (דובדבן), תואר הנדסת תוכנה, 10 שנות ניסיון
  • ✅ דף VSL מקיף — FAQs, עדויות לקוחות, השוואות, CTA
  • ✅ JSON-LD מותאם אישית — Organization schema מורחב עם Code Snippets (founder, כתובת, כתבות)

🗓️ תוכנית פעולה

שבוע 1 — דחוף

פתיחת הדלת ל-AI

  • תיקון robots.txt — ביטול חסימת AI crawlers ב-Cloudflare
  • הוספת Content-Signal: ai-input=yes
  • תיקון bug של OAI-SearchBot
  • ביטול חסימת RSS feed
  • Bing Webmaster Tools — הרשמה + submit sitemap
שבוע 2 — תשתית

חיזוק Schema + עדכניות

  • Organization schema מורחב — founder, telephone, email, address, description
  • Article/BlogPosting schema על כל הפוסטים
  • עדכון dateModified על כל הפוסטים הקיימים
  • התקנת IndexNow plugin
  • משפט ראשון הגדרתי בדפים מרכזיים
שבוע 3-4 — תוכן

תוכן חדש + Brand KB

  • דף Brand Knowledge Base — עובדות, מחירים, הבדלים, ביצועים
  • "5 אסטרטגיות מסחר בביטקוין 2026" — Best-of listicle
  • 2-3 פוסטים חדשים עם סטטיסטיקות ייחודיות
  • 2 Press Releases דרך AB Newswire (~$80 כל אחד)
חודש 2+ — צמיחה

YouTube + מדידה

  • 3-5 סרטוני YouTube על נושאים מהבלוג
  • בדיקה שבועית — לשאול ChatGPT/Perplexity/Gemini שאלות ולעקוב
  • עדכון תוכן חודשי (minimum רבעוני)
  • מעקב תנועת referral מ-AI בGoogle Analytics

💡 תובנות מפתח מהמחקר

  • 📊 רשימות "Best-of" = 3-5x יותר ציטוטים ממאמרים רגילים. חברות שמות את עצמן #1 ברשימה שלהן — AI קולט את זה.
  • 📰 Press Releases = הדרך הכי זולה להיכנס לציטוטי AI. $6-80 per release. ציטוטים מתחילים תוך 14-21 ימים.
  • 🎥 YouTube = האתר הכי מצוטט ב-AI search. Perplexity ממוצע 6.61 ציטוטים לתשובה, מעדיף YouTube. עוקף Reddit.
  • 🔢 2-3 סטטיסטיקות לכל 300 מילים = 2.1x יותר ציטוטים. מספרים מדויקים (58.5% ולא ~60%) עובדים טוב יותר.
